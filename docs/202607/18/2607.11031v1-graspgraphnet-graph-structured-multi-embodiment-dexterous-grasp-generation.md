## 摘要
跨机器人手的灵巧抓取生成具有挑战性，因为手在运动学拓扑、驱动维度和原生命令空间方面存在差异。我们引入了GraspGraphNet，一个拓扑感知的抓取生成框架，它将每只手表示为从URDF导出的运动学图，并直接生成可执行的手掌姿态和关节配置。GraspGraphNet结合了分层物体表面编码、可微分正向运动学和动态世界边缘消息传递，以建模演变的机器人-物体交互。它直接在可执行的手掌姿态和关节状态空间中应用条件流匹配，避免了后处理优化、逆运动学和重定向。使用在Barrett Hand、Allegro Hand和Shadow Hand上训练的共享模型，GraspGraphNet在40个物体的基准测试中达到了平均83.48%的成功率，每次抓取的推理时间为40毫秒。无需重新训练，同一模型在受控手指移除变体上实现了72.70%的成功率，显示出对手部拓扑变化的鲁棒性。这些结果表明，图结构的手部表示可以有效地支持不同运动学结构的机器人手的灵巧抓取生成。项目主页: https://lysees.github.io/graspgraphnet-page

## Abstract
Dexterous grasp generation across robot hands is challenging because hands differ in kinematic topology, actuation dimensions, and native command spaces. We introduce GraspGraphNet, a topology-aware grasp generation framework that represents each hand as a URDF-derived kinematic graph and directly generates executable palm poses and joint configurations. GraspGraphNet combines hierarchical object surface encoding, differentiable forward kinematics, and dynamic world-edge message passing to model evolving robot-object interactions. It applies conditional flow matching directly in executable palm-pose and joint-state space, avoiding post-processing optimization, inverse kinematics, and retargeting. Using a shared model trained on Barrett Hand, Allegro Hand, and Shadow Hand, GraspGraphNet achieves an average success rate of 83.48% with 40ms inference time per grasp on a 40-object benchmark. Without retraining, the same model achieves 72.70% success on controlled finger-removal variants, demonstrating robustness to hand-topology variations. These results suggest that graph-structured hand representations can effectively support dexterous grasp generation across robot hands with different kinematic structures. ## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **手部表示**：从URDF构建运动学图 \( G_R = (V_R, E_R) \)，节点为物理连杆（去除虚拟/未用连杆），边为关节（含类型、轴、限位、固定变换）。该图随手型自然变化大小和连通性。
- **可执行状态定义**：抓取状态 \( \mathbf{x} = [\mathbf{t}_{\text{palm}}, \mathbf{R}_{\text{palm}}^{\text{6D}}, \mathbf{q}_{\text{norm}}] \in \mathbb{R}^{9+J} \)，包含掌心平移、6D连续旋转表示、归一化关节角度（J为手部活动关节数）。状态维度随手型不同而变化。
- **条件流匹配**：学习一个时间条件速度场 \( \mathbf{v}_\theta(\mathbf{x}_t, t) \)，从张开手初始状态 \( \mathbf{x}_0 \) 向目标抓取状态 \( \mathbf{x}_1 \) 演化，通过少量Euler步（K=3）直接生成最终可执行抓取。

### 关键技术细节
1. **物体表示**：从物体表面采样1024点，构建四级层次图（1024→256→64→32点），使用最远点采样和k-NN图进行消息传递。64点特征作为细粒度特征 \( O_f \)，32点特征作为粗粒度特征 \( O_c \)，平均池化得全局描述子。
2. **机器人状态编码**：
   - 对当前状态 \( \mathbf{x}_t \)，通过可微分正运动学计算每个连杆的世界位姿 \( \mathbf{T}_t^i \)。
   - 每个连杆节点特征：连杆几何嵌入（从采样点通过PointNet编码）+ 当前位置/6D方向 + 深度嵌入 + 指尖标志 + 关联关节状态（若无则填0）。通过MLP投影到节点嵌入空间。
   - 使用3层边条件消息传递神经网络（MPNN）沿URDF运动学边传播信息，消息条件化为边上的关节属性。
3. **物体-机器人融合**：
   - **粗粒度交叉注意力**：机器人节点嵌入作为查询，粗对象特征作为键值，提供全局形状上下文。
   - **动态世界边消息传递**：在每个流步，对每个连杆节点，用k-NN在细粒度对象点中找到最近的 \( K_W \) 个点（本文 \( K_W=8 \)），构建动态世界边。边消息结合对象特征、机器人嵌入、相对位移和距离，通过残差更新层聚合。该机制随抓取演化动态更新邻近关系。
4. **速度场预测与损失函数**：
   - 指定出度最高的节点作为“掌心枢纽节点”，其嵌入输入掌姿头预测 \( \mathbf{v}_{\text{palm}} \)；共享关节头应用到每个活动关节的子连杆嵌入，预测 \( \mathbf{v}_q \in \mathbb{R}^J \)。两头部额外输入全局对象描述子和正弦时间嵌入。
   - 训练时采样 \( t \sim U(0,1) \)，线性插值初始状态和目标状态，得目标条件速度 \( \mathbf{v}^* = \mathbf{x}_1 - \mathbf{x}_0 \)。损失为MSE：\( \mathcal{L} = \text{MSE}(\mathbf{v}_{\text{palm}}, \mathbf{v}^*_{\text{palm}}) + \lambda_q \text{MSE}(\mathbf{v}_q, \mathbf{v}^*_q) \)，\( \lambda_q=1 \)。
   - 初始状态采样：目标方向附近锥角30°内随机方向，面内旋转[0,2π)均匀采样，关节置为张开。配对不同初始状态与最近的目标抓取。
5. **推理**：使用K=3步Euler积分，最终状态经转换（规范掌姿→机器人自身帧，加固定手腕偏移，去归一化关节值）得可执行命令。

## 6. 论文的主要结论与发现

1. **跨手通用性**：GraspGraphNet在Barrett、Allegro、Shadow三种手上平均成功率83.48%，超过DRO-Grasp（77.60%）和TRO-Grasp（81.00%），且推理时间仅40ms（为TRO-Grasp的1/6，DRO-Grasp的1/14）。

5. **实际部署可行**：真实世界91%成功率，从部分点云直接生成并执行。

## 7. 优点

- **直接可执行输出**：无需逆运动学或额外优化，一步到位输出掌姿+关节命令，推理极快（40ms）。
- **统一图表示**：基于URDF构建，自然封装手型拓扑和关节约束，支持不同图大小和维度，模型参数共享。
- **动态交互建模**：流每一步重新计算k-NN世界边，捕捉抓取过程中手-物几何变化，提高成功率和鲁棒性。
- **条件流匹配的运用**：在连续状态空间直接学习演化，少量步数即可获得高质量结果，避免了扩散模型的慢速采样。
- **丰富的消融和泛化实验**：对手指移除变体的评估展示了模型对结构变化的鲁棒性，具有实际意义。
