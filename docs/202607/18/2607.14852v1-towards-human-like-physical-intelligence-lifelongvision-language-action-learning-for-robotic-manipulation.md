## 摘要
类似于人类顺序学习新任务的自然能力，配备视觉-语言-动作（VLA）模型的机器人在开放世界环境中部署时应具备终身学习能力，以学习新任务。然而，最近提出的大多数终身学习模型旨在有效学习当前任务（可塑性）或在先前任务上保持高精度（稳定性），而机器人操作模型中的可塑性-稳定性权衡仍未得到充分解决。为应对这一基本挑战，我们提出了一种面向机器人操作的高效缓存终身视觉-语言-动作学习框架（即LifelongVLA），该框架通过双时间尺度自适应机制缓解可塑性-稳定性权衡，同时通过高效缓存重放策略实现低成本的机器人部署。具体而言，我们提出了一个双时间尺度LoRA门控模块，将VLA自适应分解为两个轻量级路径：用于可塑性的短期适配器和用于稳定巩固的长期适配器。这些路径通过任务感知门集成，实现对可塑性-稳定性权衡的显式控制。在技能重放阶段，提出了一种高效缓存的随机重放策略，无需存储完整轨迹即可保留更平衡的保留信号。最后，实验表明，LifelongVLA优于现有基线，展示了高效的技能扩展、对已学习操作行为的稳健保留，以及在实际xArm机器人部署中对再训练的依赖降低。

## Abstract
Similar to the natural capabilities of humans to sequentially learn new tasks, robots with Vision-Language-Action (VLA) models should possess lifelong learning ability to learn a new task when deployed in open-world environments. However, most recently proposed lifelong learning models aim to effectively learn the current task (plasticity) or maintain high accuracy on previous tasks (stability), while the plasticity-stability trade-off remains largely unsolved in robotic manipulation models. To address this fundamental challenge, we propose a cache-efficient lifelong Vision-Language-Action learning framework for robotic manipulation (i.e., LifelongVLA), which alleviates the plasticity-stability trade-off with a dual-timescale adaptation mechanism while achieving low-cost robotic deployment with a cache-efficient replay strategy. More concretely, we propose a dual-timescale LoRA gating module to decompose VLA adaptation into two lightweight pathways: a short-term adapter for plasticity and a long-term adapter for stable consolidation. These pathways are integrated via a task-aware gate, enabling explicit control of the plasticity-stability trade-off. In the skill replay phase, a cache-efficient stochastic replay strategy is proposed to preserve more balanced retention signals without full-trajectory storage. Finally, experiments show that LifelongVLA outperforms existing baselines, demonstrating efficient skill expansion, robust retention of learned manipulation behaviors, and reduced reliance on retraining for real-world deployment on an xArm robot.

## 1. 论文的核心问题与整体含义（研究动机和背景）

当前主流的视觉-语言-动作（VLA）机器人操作策略通常采用静态离线训练范式：在固定预定义技能集上一次性训练后部署，不再更新。然而，机器人在开放世界环境中持续遇到新物体、新指令、新场景布局和新动作分布，这种静态范式难以适应。机器人需要像人类一样具备**终身学习（Lifelong Learning）**能力——在获取新技能的同时保持已学技能。但终身学习面临核心挑战：**可塑性（plasticity）与稳定性（stability）的权衡**。高可塑性有助于快速学习新任务，但可能覆盖已学行为；高稳定性保留旧技能，但可能限制未来学习。现有方法（如经验重放）虽然缓解了灾难性遗忘，但存储完整轨迹开销巨大，且未充分解决可塑性-稳定性矛盾。论文旨在提出一种低成本、高效的终身VLA学习框架，实现可塑性与稳定性的显式平衡。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
LifelongVLA建立在冻结的VLA骨干网络上，通过**双时间尺度LoRA门控**和**缓存高效随机重放**两个模块，在有限重放下平衡可塑性与稳定性。

### 关键技术细节

#### 2.1 双时间尺度LoRA门控（Dual-Timescale LoRA Gating）
- **短期路径（Short-term LoRA）**：仅由当前任务损失（\(L_{new}\)）更新，提供高可塑性以快速学习新技能。
- **长期路径（Long-term LoRA）**：仅由重放损失（\(L_{rep}\)）和蒸馏损失（\(L_{dist}\)）更新，学习率较小（\(\eta_{lg} < \eta_{sh}\)），用于巩固已学技能。
- **任务感知门控（Task-aware Gate）**：从冻结的骨干前缀特征计算门控上下文 \(c\)，预测长期路径权重 \(\alpha\)，实现权重级组合：
  \[
  \Delta \widetilde{W}_t^\ell(c) = (1 - \alpha_t^\ell(c)) \Delta W_{sh,t}^\ell + \alpha_t^\ell(c) \Delta W_{lg,t}^\ell
  \]
  推理仅需单次前向传播。

#### 2.2 缓存高效随机重放（Cache-Efficient Stochastic Replay）
- **技能级前缀缓存**：每个任务随机选择少量样本（如 \(c_t=50\)），仅存储**停止梯度的前缀标记**（latent prefix tokens）和紧凑的状态-动作监督，不存储原始图像、语言token或完整轨迹，大幅降低存储开销（从167.62 MiB/任务降至95.70 MiB/任务）。
- **随机重放与后缀重新计算**：重放时重新采样扩散噪声和扩散时间步，用当前模型重新计算后缀标记，避免缓存特征与当前模型不匹配。旧缓存前缀作为技能记忆锚点，重放时产生多样化的去噪实例。
- **掩码目标函数**：将当前样本和重放样本拼接，分别计算归一化的当前任务损失和重放损失，并加入教师蒸馏损失（使用上一任务快照），最终总损失：
  \[
  L_t = L_{new} + \lambda L_{rep} + \beta L_{dist}
  \]
