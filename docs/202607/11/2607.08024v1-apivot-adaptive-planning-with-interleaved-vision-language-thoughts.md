## 摘要
长视程机器人规划需要对语义任务结构和几何可行性进行联合推理。为了成功执行任务，机器人必须分解目标、选择与任务相关的物体并对动作进行排序，同时确保规划满足空间约束，例如有限自由空间和物体碰撞。在本文中，我们提出了APIVOT，一种基于视觉语言模型（VLM）的规划器，能够自适应地交织语言和视觉思维以进行长视程规划。APIVOT学会利用语言进行语义推理，同时使用视觉思维作为想象的未来状态，对几何可行性进行内部验证。在长视程厨房任务上，APIVOT优于通用VLM和先前的规划框架，在空间受限的情况下取得了最大的改进。我们发现APIVOT学习到了有意义的模态选择行为，表明自适应地交织视觉-语言思维既提高了规划成功率也提高了推理效率。

## Abstract
Long-horizon robot planning requires jointly reasoning over semantic task structure and geometric feasibility. To successfully execute a task, a robot must decompose goals, select task-relevant objects, and sequence actions, while ensuring that plans satisfy spatial constraints such as limited free space and object collisions. In this work, we propose APIVOT, a VLM-based planner that adaptively interleaves language and visual thoughts for long-horizon planning. APIVOT learns to leverage language for semantic reasoning, while using visual thoughts as imagined future states for internal verification of geometric feasibility. On long-horizon kitchen tasks, APIVOT outperforms general-purpose VLMs and prior planning frameworks, achieving the largest gains in spatially constrained settings. We find that APIVOT learns meaningful modality selection behavior, demonstrating that adaptive interleaving of vision-language thoughts improves both planning success and reasoning efficiency.

### 2. 论文提出的方法论

*   **核心思想**:
    *   **APIVOT**（Adaptive Planning with Interleaved Vision-Language Thoughts）是一种基于VLM的规划器，其核心是生成一个**多模态推理轨迹**，在语言推理步骤中，**根据需求动态插入“视觉思维”**。这些视觉思维被模型编码为**潜在视觉状态**，用以表征想象的未来场景，从而在执行前在内部完成几何可行性验证。

*   **关键技术细节**:
    *   **视觉思维实现**:
        *   视觉思维由一段特殊的离散视觉符元（visual tokens）表示，形式为 `<|image_start|><|image_pad|>...<|image_pad|><|image_end|>`。这些离散符元标记了视觉思维在自回归序列中的位置。
        *   对应这些视觉符元的解码器隐状态（hidden states）被用作**潜在视觉状态** `H_j` ，它编码了模型对“未来状态”的想象。
    *   **视觉对齐**:
        *   为了保证想象的准确性，在训练时，将一个步骤对应的**真实未来图像** `I_gt_j` 通过冻结的视觉编码器得到目标特征 `Z_gt_j`。
        *   模型生成的潜在视觉状态 `H_j` 会通过一个投影头 `f_ϕ` 映射到与 `Z_gt_j` 相同的特征空间，并计算两者间的**余弦相似度损失** `L_vis`，以监督想象内容。
    *   **三阶段训练课程**:
        *   **第一阶段：视觉思维理解。** 在每个推理步骤都提供**真实的潜在视觉特征**给模型，训练模型学会如何利用这些视觉信息进行下游规划。
        *   **第二阶段：视觉思维生成。** 在每个推理步骤都要求模型**自主生成**潜在视觉状态，并通过视觉对齐损失 `L_vis` 进行优化，教模型学会如何想象。
        *   **第三阶段：自适应模态选择。** 训练数据的推理轨迹中，只在几何约束紧张的步骤才包含视觉思维。模型学习**何时生成**视觉思维（仅当必要时），何时仅用语言推理。
    *   **总体损失函数**:
        *   `L_total = L_CE,plan + λ_CE,trace * L_CE,trace + λ_vis * L_vis`
        *   其中 `L_CE,plan` 和 `L_CE,trace` 是用于监督最终计划和推理轨迹的交叉熵损失，`L_vis` 是视觉对齐损失。

*   **视野表征瓶颈**: 模型生成的仅是潜在视觉状态，其表达能力和准确性受限于训练数据和模型容量，可能无法捕捉真实世界中多样化的几何和外观变化。实验结果显示，与使用“真实未来图像”的oracle上限相比仍有明显差距。
*   **模态选择策略可提升空间**: 当前的自适应模态选择是通过带有启发式标注的监督微调学习而来，而非通过强化学习等直接面向任务成功的优化。消融实验也表明该策略对监督信号的密度敏感，其鲁棒性有待增强。
*   **表示模态单一**: 目前推理仅在文本和潜在图像之间切换，尚未探索加入点、边界框等更为简洁明确的视觉原语（visual primitives），这可能限制规划效率和精度的进一步提升。
